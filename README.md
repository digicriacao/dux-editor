# Gerador de Receitas DUX

Ferramenta interna para gerar HTML responsivo de receitas no padrão visual DUX.

## Acesso

A ferramenta está hospedada em GitHub Pages. Acesse pela URL do repositório (ver Settings → Pages).

## Como usar

1. Abra a ferramenta no navegador
2. Preencha os dados da receita (dificuldade, preparo, rendimento, ingredientes, modo de preparo)
3. Use os botões de **Negrito (B)** e **Link (🔗)** no editor de modo de preparo
4. Clique em **Copiar HTML** e cole na plataforma de site

O HTML gerado é 100% inline (CSS em cada elemento), compatível com VTEX, Shopify, WordPress, Wix, e-mail marketing.

## Estrutura

```
index.html       — a ferramenta
fontes/          — arquivos .woff2 da fonte DUXText (Regular, Bold, Light, Italic)
```
