Gerador de Receitas DUX

Gera HTML 100% inline — funciona em qualquer plataforma de site, e-commerce ou e-mail.

Dados da receita
Já preenchido como fontes/ (caminho relativo) — funciona se as fontes estiverem na pasta fontes/ ao lado da ferramenta no GitHub Pages. Para uso fora do GitHub, cole a URL completa, ex.: https://seu-usuario.github.io/dux-receitas/fontes/. Vazio = fallback Helvetica. Salvo automaticamente no navegador.
Negrito = também sublinha (estilo DUX)

HTML 100% inline (sem <style> nem <script>) — compatível com VTEX, Shopify, WordPress, Wix, e-mail marketing e qualquer editor que sanitize HTML.
